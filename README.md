# theTuring's filehub
theTuring的文件中心，支持多格式的文件上传下载</br>
暂支持文件格式（xls,xlsx,doc,docx,ppt,pptx,pdf,jpg,jpeg,png,mp3,wav,txt,zip）</br>
在线预览支持格式（xls,xlsx,doc,docx,jpg,jpeg,png）</br>
